Paste an exported YAJA scene or ordinary bB sprite data.
  • YAJA Data Only and Compilable Demo exports restore animation settings, frames, timing, assignments, offsets, NUSIZ, pixels, and colors.
  • Ordinary bB imports accept one or two player#: blocks, with optional player#color: blocks.
  • Keep the ;@YAJA and frame marker comments intact for a complete scene round-trip.

;@YAJA FRAME_BEGIN 0 { ...frame settings... }
