# How to Run YAJA 2600 Animator on Linux

Extract the full Linux ZIP, then launch `YAJA 2600 Animator` from the extracted folder.

If your archive manager does not preserve executable permissions, open a terminal in that folder and run:

```sh
chmod -R u+x "YAJA 2600 Animator"
./"YAJA 2600 Animator"
```

Keep the executable beside its `resources`, `locales`, and library files.
